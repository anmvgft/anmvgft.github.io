# Installation

### Download the repo from:

https://github.com/anmvgft/anmvgft.github.io

### Open a terminal in the downloaded folder and run:

```bash
npm install
```

# Development

### To serve locally, use this command:

```bash
npm run serve
```

# Testing

### Run tests

```bash
npm run test
```

# Format and analysis

### Run formatter

```bash
npm run format
```

### Check dependencies

```bash
npm audit
```

### Run linter:

```bash
npm run lint
```

# Build

### To create a build in the docs folder, type:

```bash
npm run build
```

### Deploy url:

https://anmvgft.github.io
